# คู่มือการใช้งานและบำรุงรักษาคอมพิวเตอร์ (Computer/Notebook Usage and Maintenance Guide)

**ข้อมูลอ้างอิงเอกสาร:**
- Document No.: QAI-N1-SMF-090-2803
- Rev. No.: 0 
- วันที่ออก (Issued date): 24-Nov-25 
- แจกจ่ายให้ (Distribution to): Smart Factory Division 

## วัตถุประสงค์ (Purpose) 
เพื่อให้คำแนะนำเกี่ยวกับการใช้งานคอมพิวเตอร์อย่างถูกต้องและการบำรุงรักษาเพื่อยืดอายุการใช้งาน รวมถึงการป้องกันปัญหาที่อาจเกิดขึ้น 

## ประยุกต์ใช้กับ (Application) 
Smart Factory Computer/Notebook 

## รายละเอียด (Details): ข้อปฏิบัติในการใช้คอมพิวเตอร์ได้อย่างถูกต้อง 

### การเปิด-ปิดเครื่อง 
1. เสียบสายไฟของเครื่อง Uninterruptible Power Supply (UPS) เข้ากับแหล่งจ่ายไฟแล้วเปิดเครื่องเพื่อทำหน้าที่เป็นเครื่องสำรองพลังงานไฟฟ้าให้กับอุปกรณ์ต่าง ๆ เมื่อเกิดปัญหาทางไฟฟ้า (หมายเหตุ: เอกสารมีรูปภาพเครื่องสำรองไฟ UPS)
2. เชื่อมต่อสายไฟฟ้าของคอมพิวเตอร์เข้ากับเครื่องสำรองไฟ UPS
3. กดปุ่ม power เพื่อเปิดคอมพิวเตอร์ และ log in username/password ตามที่กำหนด
4. ทำงานบนระบบคอมพิวเตอร์ตามที่ได้รับมอบหมาย
5. เมื่อใช้งานคอมพิวเตอร์เสร็จแล้ว จะต้องปิดคอมพิวเตอร์ ตามขั้นตอนนี้ 
   - 5.1 คลิกที่ปุ่ม Start 
   - 5.2 เลือกคำสั่ง Shutdown จากเมนูระบบปฏิบัติการ รอสักครู่เครื่องจะดับโดยอัตโนมัติ 
   - 5.3 ปิดปุ่มหน้าจอภาพ Monitor 
   (หมายเหตุ: เอกสารมีรูปแสดงขั้นตอนการปิดเครื่องคอมพิวเตอร์ )

### สิ่งที่ต้องตรวจสอบหลังทำการเปิดเครื่องก่อนเริ่มใช้งาน 
1. Cortex XDR ต้อง Enabled 
2. ManageEngine ต้อง Active โดย Icon ต้องไม่เป็นสีเทา 
3. Windows Security ต้องขึ้น No actions needed
(หมายเหตุ: เอกสารมีรูปประกอบของทั้ง 3 หน้าจอการตรวจสอบ)

### การจัดวางและสภาพแวดล้อม
1. วางคอมพิวเตอร์บนพื้นผิวที่แข็งและเรียบ มีการระบายอากาศดี
2. หลีกเลี่ยงแสงแดดโดยตรงและความชื้นสูง
3. ควรมีอุณหภูมิห้องระหว่าง 20-25 องศาเซลเซียส
4. ไม่วางของหนักทับเครื่องหรือบริเวณช่องระบายอากาศ 
5. ใช้ UPS หรือเครื่องสำรองไฟเพื่อป้องกันไฟตก 

### การใช้งานที่ดี 
1. ไม่ควรใช้งานเครื่องติดต่อกันเกิน 8-10 ชั่วโมงโดยไม่พัก 
2. ปิดโปรแกรมที่ไม่ใช้งานเพื่อลด CPU และ RAM usage 
3. สำรองข้อมูลสำคัญเป็นประจำ 
4. ใช้ซอฟต์แวร์ที่ถูกลิขสิทธิ์และอัปเดตเป็นประจำ 

### การสำรองข้อมูล (Backup) 
1. Cloud Storage (Google Drive, OneDrive, Dropbox) 
2. External Hard Drive หรือ NAS 
3. ใช้โปรแกรม Backup อัตโนมัติ เช่น Windows Backup 

### การยืดอายุการใช้งาน 
1. ใช้ Power Saving Mode เมื่อไม่ได้ใช้งานหนัก 
2. ไม่ปล่อยให้แบตเตอรี่ (Laptop) หมดจนเกินไป 
3. ถอดแบตเตอรี่ออกหากใช้เฉพาะไฟบ้านเป็นเวลานาน 
4. หลีกเลี่ยงการกระแทกหรือสั่นสะเทือน 

### การใช้งานที่ถูกสุขภาพ 
1. จัดหน้าจอให้ห่างจากดวงตาประมาณ 50-70 ซม.
2. ใช้กฎ 20-20-20: ทุก 20 นาที มองของห่าง 20 ฟุต เป็นเวลา 20 วินาที 
3. ปรับความสูงของเก้าอี้และคีย์บอร์ดให้เหมาะสม 
4. พักเบรกทุก 1-2 ชั่วโมง 
(หมายเหตุ: เอกสารมีภาพประกอบวิธีนั่งใช้งานคอมพิวเตอร์ที่ถูกต้องและเคล็ดลับบำรุงสายตา)

### การ Maintenance คอมพิวเตอร์ 
**1. การทำความสะอาดฮาร์ดแวร์** 
- ทำความสะอาดฝุ่น 
- ทำความสะอาดพัดลม ของ CPU และการ์ดจอเพื่อระบายความร้อนได้ดี 
- ตรวจสอบช่องระบายอากาศ ไม่ให้อุดตัน 

**2. การบำรุงรักษาซอฟต์แวร์** 
- อัปเดต OS และโปรแกรม เป็นประจำเพื่อความปลอดภัย 
- Disk Cleanup ลบไฟล์ขยะและไฟล์ชั่วคราว
- Backup ข้อมูล อย่างสม่ำเสมอ 

**3. การบำรุงรักษา UPS** 
- **3.1. ตรวจสอบแบตเตอรี่** 
  - ทดสอบแบตเตอรี่ ทุก 3-6 เดือน โดยถอดปลั๊กไฟเพื่อทดสอบว่าใช้งานได้นานเท่าใด 
  - เปลี่ยนแบตเตอรี่ 2 ปี หรือตามคำแนะนำของผู้ผลิต 
  - ตรวจสอบสัญญาณเตือน จาก UPS เช่น เสียงปี๊บหรือไฟแจ้งเตือน 
- **3.2. ตรวจสอบการทำงาน** 
  - ตรวจสอบแรงดันไฟ output ว่าเสถียร 
  - ตรวจสอบเสียง ผิดปกติจากตัวเครื่อง 
- **3.3. การทำความสะอาด** 
  - เช็ดฝุ่นตัวเครื่อง UPS 
  - ทำความสะอาดช่องระบายอากาศ 
  - ตรวจสอบสายไฟและปลั๊กไม่ให้ชำรุด
